Aegis

Team: House of Brahma

Aegis is an AI-powered online exam monitoring platform built for the SYCET IGNITE Hackathon 2026.

The goal of this project is to make online assessments more secure and reliable by using realtime monitoring and browser-based AI detection. Aegis tracks suspicious activities during an exam session and provides live alerts, risk analysis, and monitoring reports through a modern dashboard interface.

The platform is designed to simulate how a real-world remote proctoring system works while keeping the experience lightweight, fast, and easy to use.

Project Vision

With the rise of remote learning and online assessments, maintaining exam integrity has become a major challenge. Aegis aims to provide a smarter and more practical approach to online proctoring using browser-based AI and realtime monitoring systems.

This project focuses on creating a solution that feels modern, reliable, and scalable while remaining efficient enough to run directly in the browser.
